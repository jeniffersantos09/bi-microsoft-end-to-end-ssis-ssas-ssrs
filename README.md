# bi-microsoft-end-to-end-ssis-ssas-ssrs

        Este projeto apresenta uma solução de Business Intelligence construída utilizando a stack Microsoft (SQL Server, SSIS, SSAS e SSRS), cobrindo todas as etapas essenciais de um ambiente corporativo de dados — desde o banco operacional até a disponibilização de estruturas analíticas e relatórios.
1. Banco Relacional (OLTP)
Modelagem do ambiente transacional
Criação das tabelas operacionais
Geração de dados com T-SQL
2. Área de Stage
Camada intermediária para tratamento
Cargas iniciais via SSIS
3. Data Warehouse
Modelagem dimensional
Criação de tabelas fato e dimensão
Dimensões com histórico (SCD)
Carga incremental com técnicas de watermark
4. ETL com SSIS
Pipelines de extração, transformação e carga
Tratamento de exceções
Normalização e controle de chaves substitutas
-Componentes Adicionais em Expansão
Publicação de cubo no SSAS
Dashboards analíticos
Relatórios corporativos com SSRS

Otimizações de performance
